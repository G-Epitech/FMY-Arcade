## Installation
### Cmake
```
sudo apt install cmake
```
More informations [here](https://cmake.org/download/).

### Pre-Commit
First, install pre-commit package.
- On linux:
```
pip install pre-commit
```
- On macOS (Intel/AMD):
```
brew install pre-commit
```
Next, steup pre-commit at the root of the repository:
```
pre-commit install
```

## Build project
Build the project
```
make
```
Clean the project
```
make fclean
```
